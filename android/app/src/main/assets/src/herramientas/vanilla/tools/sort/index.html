De Menor a Mayor

info Qué es

Un ejercicio de memoria de trabajo y cálculo mental. Aparecen 3 valores que tienes que memorizar. Después desaparecen y tienes que decidir si estaban ordenados de menor a mayor.

tune Niveles

  • Números: solo tienes que recordar y comparar números.
  • Sumas y restas: tienes que resolver las operaciones, retener los resultados y compararlos.
  • Multiplicaciones: el más exigente — cálculo rápido bajo presión.

play_circle Cómo funciona

  1. Pulsa INICIAR. Aparecen 3 valores — memórizalos.
  2. Los valores desaparecen. ¿Estaban ordenados de menor a mayor?
  3. Pulsa VERDADERO si sí, o FALSO si no.

lightbulb Consejos

  • Empieza con números simples para coger el ritmo.
  • Las operaciones son mucho más difíciles — no te agobies si fallas.
  • Mira tu tiempo de reacción medio para ver si mejoras.
  • Combínalo con ejercicio físico para añadir dificultad.

fitness_center Cuándo aplicar

  • Como doble tarea: calcula mientras haces ejercicio aeróbico.
  • Para mantener las habilidades numéricas.
  • Para entrenar cognitivamente bajo fatiga física.
  • En grupo: todos responden a la vez con gestos.
