Memoria Visual

info Qué es

El clásico juego de emparejar cartas ocultas. Voltea dos cartas por turno e intenta encontrar todas las parejas. Entrena tu memoria espacial y tu atención.

play_circle Cómo funciona

  1. Elige nivel: fácil (3 parejas), medio (6) o difícil (10).
  2. Pulsa COMENZAR. Las cartas se barajan.
  3. Toca una carta para voltearla. Luego toca otra.
  4. Si coinciden, quedan descubiertas. Si no, se ocultan de nuevo.

lightbulb Consejos

  • Intenta recordar dónde estaba cada carta que ya viste.
  • Añade una regla motora: “haz 3 sentadillas entre cada turno”.
  • Intenta bajar el número de intentos en cada partida.
  • 10 parejas es un reto real — no te frustres si tardas.

fitness_center Cuándo aplicar

  • Como pausa activa entre series de fuerza.
  • En parejas o grupo: cada persona voltea una carta por turno.
  • Para entrenar la memoria de forma divertida (cualquier edad).
  • Como vuelta a la calma con componente lúdico.
