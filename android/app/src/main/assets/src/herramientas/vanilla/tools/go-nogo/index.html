Go / No-Go

info Qué es

Un ejercicio de control inhibitorio: tienes que reaccionar rápido cuando aparece una señal, pero frenar cuando aparece otra. Entrena tu capacidad de decidir bajo presión.

play_circle Cómo funciona

  1. Pulsa INICIAR. Irán apareciendo círculos de colores.
  2. Verde = toca la pantalla lo más rápido posible.
  3. Rojo = no toques nada, frena el impulso.
  4. Escucharás un sonido cada vez que cambie el color.
  5. Puedes tocar en cualquier parte de la pantalla.

tune Niveles

  • Básico: solo verde (toca) y rojo (frena). Ideal para empezar.
  • Distractores: aparecen colores extra (amarillo, azul) que NO debes tocar. Solo el verde.
  • Invertido: las reglas se invierten — rojo = toca, verde = frena. Rompe el automatismo.

lightbulb Consejos

  • Empieza en básico para coger el ritmo.
  • Si puedes, mueve el cuerpo entre toques (sentadillas, desplazamientos).
  • Vigila los “falsos”: si suben mucho, tu cerebro está cansado.
  • El modo invertido es difícil — no te frustres, es normal fallar.

fitness_center Cuándo aplicar

  • Como calentamiento cognitivo con alta activación.
  • Para trabajar el control de impulsos bajo presión.
  • Si quieres medir tu fatiga cognitiva (si los falsos suben, estás cansado).
  • En grupo: todos tocan la misma pantalla o cada uno la suya.
