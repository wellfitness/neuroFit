Sonidos Audiomotores

info Qué es

Un ejercicio de reacción auditiva. Escuchas sonidos de distintas frecuencias (grave, agudo, medio) y tienes que hacer el movimiento asociado a cada uno. Entrena tu oído y tu velocidad de reacción.

play_circle Cómo funciona

  1. Cada sonido tiene un movimiento asignado. Puedes cambiarlos editando los campos de texto.
  2. Pulsa INICIAR. Las primeras rondas te muestran qué hacer como ayuda.
  3. Después, el texto desaparece y tendrás que recordar las asociaciones.
  4. Ajusta la cadencia para subir o bajar la dificultad.

lightbulb Consejos

  • Puedes hacer este ejercicio sin mirar la pantalla — incluso de espaldas.
  • Cambia los movimientos entre series para añadir dificultad extra.
  • Si fallas mucho, baja la cadencia — no pasa nada.
  • Cierra los ojos para máximo reto — solo te queda el sonido.

fitness_center Cuándo aplicar

  • Para trabajar doble tarea: haz ejercicio mientras reaccionas al sonido.
  • Si quieres entrenar sin depender de la vista.
  • En deportes donde las señales auditivas son clave (silbatos, indicaciones).
  • Como vuelta a la calma con reactividad baja.
