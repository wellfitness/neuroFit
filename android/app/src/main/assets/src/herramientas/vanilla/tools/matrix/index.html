Matriz Visoespacial

info Qué es

Un ejercicio de memoria visoespacial. Se iluminan celdas en una cuadrícula y tienes que recordar cuáles eran. Entrena tu capacidad de retener posiciones en el espacio.

play_circle Cómo funciona

  1. Elige el tamaño de la cuadrícula (3x3, 4x4 o 5x5).
  2. Pulsa INICIAR. Se iluminan varias celdas un momento.
  3. Cuando te pida respuesta, toca las celdas que recuerdes.
  4. Con “Modo orden” activado, tienes que tocarlas en el mismo orden en que aparecieron.

lightbulb Consejos

  • Empieza en 3x3 sin modo orden para coger el ritmo.
  • El modo orden es mucho más difícil — pruébalo cuando domines el básico.
  • Combina con equilibrio o sentadillas durante la memorización.
  • 5x5 con modo orden es un reto extremo — no te frustres.

fitness_center Cuándo aplicar

  • Para entrenar la memoria espacial (posiciones, ubicaciones).
  • Como doble tarea con carga espacial durante ejercicio físico.
  • En rehabilitación cognitiva.
  • Para mantener la mente activa (cualquier edad).
