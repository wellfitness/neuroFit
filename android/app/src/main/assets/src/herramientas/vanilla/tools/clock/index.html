Reloj Auditivo (ACT)

record_voice_over Audio por voz

Esta herramienta usa la voz del sistema. Si no oyes nada:

  1. Ve a Ajustes → Accesibilidad → Texto a voz.
  2. Instala o activa un motor en español (recomendado: «Motor de texto a voz de Google»).
  3. Sube el volumen de medios.

info Qué es

Un ejercicio de razonamiento espacial. Se anuncia una hora y tienes que imaginar un reloj analógico: ¿las dos agujas caen del mismo lado o en lados opuestos? Parece fácil, pero es un reto real.

play_circle Cómo funciona

  1. Elige variante: Izq/Der (línea 12-6) o Arriba/Abajo (línea 3-9).
  2. Pulsa INICIAR. Se anuncia una hora aleatoria.
  3. Imagina el reloj: ¿las agujas están del mismo lado?
  4. Pulsa Mismo lado u Opuestos.

lightbulb Consejos

  • Es más difícil de lo que parece — empieza en cadencia lenta.
  • Dibuja el reloj con el dedo en el aire si te ayuda a visualizarlo.
  • Necesitas saber leer relojes analógicos para este ejercicio.
  • No te frustres si fallas — la rotación mental es exigente.

fitness_center Cuándo aplicar

  • Para entrenar razonamiento espacial y rotación mental.
  • Como doble tarea avanzada mientras haces ejercicio.
  • Para mantener las funciones ejecutivas activas.
  • Para medir la agilidad mental antes y después de entrenar.
