Listas de Objetos

info Qué es

Un ejercicio de memoria de trabajo. Van apareciendo objetos (animales o alimentos) uno a uno. Cuando desaparecen, tienes que recordar si estaban ordenados por tamaño.

tune Niveles

  • 1 Lista: objetos de una sola categoría. ¿Iban de menor a mayor tamaño?
  • 2 Listas: objetos mezclados de dos categorías. ¿Iban primero los alimentos y luego los animales, ambos de menor a mayor?
  • Inverso: como 2 Listas, pero el orden va de mayor a menor. El más difícil.

play_circle Cómo funciona

  1. Elige nivel, cantidad de objetos (3-5) y cadencia.
  2. Pulsa INICIAR. Los objetos aparecen uno a uno — memórizalos.
  3. Desaparecen. ¿Estaban en el orden correcto?
  4. Pulsa VERDADERO o FALSO.

lightbulb Consejos

  • Empieza con 1 Lista y 3 objetos para coger el ritmo.
  • 2 Listas es mucho más difícil — tienes que clasificar y ordenar a la vez.
  • Combina con marcha, equilibrio o sentadillas durante la memorización.
  • Mira tu tiempo de reacción medio para ver si mejoras.

fitness_center Cuándo aplicar

  • Para entrenar memoria de trabajo y capacidad de ordenación mental.
  • Como doble tarea: memoriza mientras haces ejercicio aeróbico.
  • Para mantener la memoria activa (especialmente en adultos mayores).
  • En rehabilitación cognitiva: retención y categorización.
