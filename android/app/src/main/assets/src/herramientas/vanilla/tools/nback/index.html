N-Back Visual

info Qué es

Uno de los mejores ejercicios para entrenar la memoria de trabajo. Van apareciendo formas y tienes que detectar cuándo la actual es igual a la de hace N turnos.

tune Niveles

  • Fácil: 1-Back, solo forma. ¿Es igual a la anterior?
  • Medio: 1-Back, forma + color. Dos cosas que recordar.
  • Difícil: 2-Back. ¿Es igual a la de hace 2 turnos? Muy exigente.

play_circle Cómo funciona

  1. Pulsa INICIAR. Van apareciendo formas de colores.
  2. Si la forma actual es igual a la de hace N turnos, pulsa COINCIDE.
  3. Si no coincide, no toques nada — espera a la siguiente.
  4. El historial de abajo te ayuda a recordar la secuencia.

lightbulb Consejos

  • Empieza en Fácil para entender la mecánica.
  • Combina con caminata, bici estática o equilibrio para más dificultad.
  • El 2-Back es muy difícil — es normal fallar mucho al principio.
  • Si las omisiones suben, tu memoria está cansada.

fitness_center Cuándo aplicar

  • Para entrenar específicamente la memoria de trabajo.
  • Como doble tarea cognitivo-motora de alta demanda.
  • Para medir tu capacidad cognitiva antes y después de entrenar.
  • Para mantener la mente activa (especialmente con la edad).
