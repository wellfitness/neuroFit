Temporizadores de Entrenamiento

info Qué es

4 temporizadores para distintos formatos de entrenamiento. Elige el modo que mejor se adapte a tu sesión.

avg_pace EMOM

Every Minute On the Minute. Cuenta atrás por intervalo configurable (60s, 90s, 120s...). Al llegar a 0, suena la alerta y comienza la siguiente ronda. Soporta EM90M, EM2M, etc.

sync Intervalos

Trabajo / Descanso. Alterna fases de trabajo y descanso durante las rondas configuradas. Perfecto para Tabata, HIIT o cualquier formato de intervalos.

speed AFAP

As Fast As Possible. Cronómetro ascendente que mide cuánto tardas en completar un WOD. Tiene un límite de tiempo configurable.

local_fire_department AMRAP

As Many Rounds As Possible. Cuenta atrás desde la duración configurada. Completa tantas rondas como puedas antes de que termine el tiempo.

lightbulb Consejos

  • Configura el tiempo de preparación para tener margen antes de empezar.
  • En grupo: pon la tablet donde todos puedan ver el tiempo.
  • La pantalla no se apagará mientras el timer esté activo.
  • Activa o desactiva el sonido según dónde estés.
