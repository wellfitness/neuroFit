Búsqueda Visual

info Qué es

Un ejercicio de atención visual. Tienes que encontrar una letra diferente escondida entre muchas letras parecidas. Entrena tu velocidad de rastreo visual y tu capacidad de filtrar distracciones.

play_circle Cómo funciona

  1. Pulsa INICIAR. Aparece una cuadrícula llena de letras.
  2. Encuentra la letra diferente y tócala lo más rápido posible.
  3. Arriba te dice cuál es la letra que buscas.
  4. Si no la encuentras a tiempo, cuenta como error.

lightbulb Consejos

  • No escanees letra por letra — intenta ver la cuadrícula entera.
  • Empieza en cadencia lenta para coger el truco.
  • Combina con ejercicio: busca la letra mientras caminas o pedaleas.
  • Si los errores suben, tu vista o tu cerebro están cansados.

fitness_center Cuándo aplicar

  • En deportes con alta demanda visual (raqueta, portería).
  • Como doble tarea: buscar mientras haces ejercicio.
  • Para medir fatiga visual durante sesiones largas.
  • Como calentamiento cognitivo antes de actividades de precisión.
