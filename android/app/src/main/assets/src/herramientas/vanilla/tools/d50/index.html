Decisión D50

info Qué es

Un ejercicio de cálculo mental rápido. Aparecen dos números, los sumas mentalmente y decides si el resultado es mayor o menor que 50. Fácil de entender, difícil bajo presión.

play_circle Cómo funciona

  1. Pulsa INICIAR. Aparecen dos números.
  2. Súmalos mentalmente.
  3. Pulsa MAYOR si la suma supera 50, o MENOR si no llega.

lightbulb Consejos

  • Solo necesitas saber sumar — es un buen ejercicio para empezar.
  • Combina con caminata, bici o steps para añadir dificultad.
  • Sube la velocidad cuando te sientas cómodo.
  • En grupo: todos responden en voz alta (“Mayor” / “Menor”).

fitness_center Cuándo aplicar

  • Como calentamiento cognitivo rápido y accesible.
  • Como doble tarea: calcula mientras haces ejercicio.
  • Para mantener el cálculo mental activo.
  • Para medir tu estado cognitivo antes y después de entrenar.
