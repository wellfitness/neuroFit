Go / No-Go

info Qué es

Un ejercicio de control inhibitorio: tienes que reaccionar rápido cuando aparece una señal, pero frenar cuando aparece otra. Entrena tu capacidad de decidir bajo presión.

play_circle Cómo funciona

  1. Pulsa INICIAR. Irán apareciendo círculos de colores.
  2. Verde = toca la pantalla lo más rápido posible.
  3. Rojo = no toques nada, frena el impulso.
  4. Escucharás un sonido cada vez que cambie el color.
  5. Puedes tocar en cualquier parte de la pantalla.

tune Niveles

  • Básico: solo verde (toca) y rojo (frena). Ideal para empezar.
  • Distractores: aparecen colores extra (amarillo, azul) que NO debes tocar. Solo el verde.
  • Invertido: las reglas se invierten — rojo = toca, verde = frena. Rompe el automatismo.
  • Sostenida 80/20: casi todo verde (80%) y de vez en cuando un rojo (20%). El cerebro se acostumbra a tocar y tiene que romper el impulso cuando aparece el rojo. Mide inhibición proactiva y post-error slowing (si tras un error te vuelves más prudente). Mejor para bloques largos de 3−5 minutos.

sports_gymnastics Ejemplos en sala

  • Sentadilla cognitiva: aguanta abajo en sentadilla isométrica. Con verde sube explosivo (plio); con rojo aguanta 3 s más.
  • Ida y vuelta reactiva: trote en sitio. Verde → carrera explosiva 3 m; rojo → quieto inmediato.
  • Saltos en sitio: bota suave. Verde → salto vertical máximo (pliométrico); rojo → baja a cuclillas y aguanta.
  • Golpeo con manopla: golpea con ritmo libre. Verde → directo potente; rojo → frena el puño antes de tocar (control excéntrico).
  • Plancha dinámica: mantén plancha. Verde → toca el hombro opuesto; rojo → aguanta inmóvil.
  • Equilibrio unipodal: de pie en una pierna. Verde → balancea la pierna libre; rojo → inmoviliza toda la postura.
  • Modo Sostenida 80/20 con trote largo: 3−5 min de trote continuo. Casi todo verde — el cliente se va acomodando — y de vez en cuando rojo. Mide el control sostenido.

record_voice_over Modo Voz

En este modo la app anuncia los colores en voz alta y la pantalla se ilumina, pero NO hay que tocar nada.

  • El entrenador define la acción: cuando suene VERDE, el cliente hace el ejercicio (sentadilla, salto, jab, paso, palmada…). Con ROJO, freeze.
  • Permite trabajar inhibición con tareas motoras de cuerpo entero, lejos del tablet.
  • Se mide cuántas rondas y la duración. La precisión la observa el entrenador.

lightbulb Consejos

  • Empieza en básico para coger el ritmo.
  • Si puedes, mueve el cuerpo entre toques (sentadillas, desplazamientos).
  • Vigila los “falsos”: si suben mucho, tu cerebro está cansado.
  • El modo invertido es difícil — no te frustres, es normal fallar.

fitness_center Cuándo aplicar

  • Como calentamiento cognitivo con alta activación.
  • Para trabajar el control de impulsos bajo presión.
  • Si quieres medir tu fatiga cognitiva (si los falsos suben, estás cansado).
  • En grupo: todos tocan la misma pantalla o cada uno la suya.
